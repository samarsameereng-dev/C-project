# C-project
A course project using C
Program processes a two-dimensional array A[N][N] with integers in the range [-1000, 1000]. It prints the task instructions and author’s name, then reads input values for A and displays the matrix. It creates a one-dimensional array C containing elements from A that are within the range [K, L], where K and L are user inputs. Array C is then sorted in ascending order, and both the unsorted and sorted versions of C are printed.
